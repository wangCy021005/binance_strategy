# 第04课：技术指标的真实角色

## 核心观点
"指标不是'买卖信号'，而是特征工程。"

---

## "金叉必涨"的幻灭

新手常犯的错误是将技术指标当作直接买卖信号。测试结果显示，单独使用MACD金叉策略，胜率约52%，扣除手续费后年化收益为-8%。原因在于：

- 指标具有**滞后性**：信号出现时价格已变动
- 指标是**简化的**：单一数字无法概括市场复杂状态
- 信号被**过度利用**：广为人知的信号会自我失效

---

## 4.1 指标的本质

### 指标 = 数学变换

| 原始数据 | 变换方式 | 得到的指标 |
|---------|---------|-----------|
| 收盘价序列 | 滑动平均 | MA |
| 收盘价序列 | 指数加权平均 | EMA |
| 涨跌幅序列 | 相对强度计算 | RSI |
| 价格序列 | 均值 ± 标准差 | 布林带 |

**核心洞察**：指标不创造新信息，只是换一种方式呈现已有信息。

### 滞后性是固有属性

| 指标 | 典型参数 | 滞后程度 |
|------|---------|---------|
| MA5 | 5日均线 | 约2-3天 |
| MA20 | 20日均线 | 约10天 |
| MACD | 12,26,9 | 约5-10天 |
| RSI(14) | 14日 | 约7天 |

---

## 4.2 趋势类指标

### MA / EMA

```
MA(5) = (P₁ + P₂ + P₃ + P₄ + P₅) / 5
EMA = 今日价格 × α + 昨日EMA × (1-α)，其中 α = 2/(N+1)
```

| 对比 | MA | EMA |
|-----|-----|-----|
| 权重分布 | 平均 | 近期更高 |
| 对新数据反应 | 慢 | 快 |
| 适用场景 | 长期趋势 | 短期趋势 |

### MACD

```
DIFF (快线) = EMA₁₂ - EMA₂₆
DEA (信号线) = EMA₉(DIFF)
柱状图 = 2 × (DIFF - DEA)   ← 中国软件惯例
```

注意：`2×`系数是国内软件惯例，国际标准（TradingView等）不含此系数。

### 背离分析

| 背离类型 | 价格表现 | MACD表现 | 含义 |
|---------|---------|---------|------|
| 顶背离 | 创新高 | 未创新高 | 上涨动能衰竭 |
| 底背离 | 创新低 | 未创新低 | 下跌动能衰竭 |

**警告**：背离只是"可能"信号，单独使用胜率通常仅55-60%。

## 4.3 振荡类指标

### RSI

```
RS = 平均上涨幅度 / 平均下跌幅度
RSI = 100 - 100/(1+RS)
```

| RSI区间 | 传统解读 | 实际情况 |
|---------|---------|---------|
| >70 | 超买 | 强势市场可能持续>80 |
| 30-70 | 正常区间 | 大部分时间在这里 |
| <30 | 超卖 | 弱势市场可能持续<20 |

### 布林带

```
中轨 = MA₂₀
上轨 = MA₂₀ + 2σ
下轨 = MA₂₀ - 2σ
```

**关键洞察**：布林带在震荡市是回归信号，在趋势市是突破信号——需先判断市场状态。

---

## 4.4 波动率指标

### 历史波动率

```
年化波动率 = 日收益率标准差 × √252
```

### ATR（Average True Range）

```
True Range = max(
    当日最高 - 当日最低,
    |当日最高 - 昨日收盘|,
    |当日最低 - 昨日收盘|
)
ATR = True Range 的 N 日平均
```

实战用途：止损距离 = 入场价 ± 2×ATR

---

## 4.5 策略评价指标

### Sharpe Ratio

```
Sharpe Ratio = (Rₚ - Rᶠ) / σₚ
```

| Sharpe值 | 评价 |
|---------|------|
| <0 | 亏损 |
| 0-1 | 一般 |
| 1-2 | 良好 |
| 2-3 | 优秀 |
| >3 | 顶级（或数据有问题） |

### 其他风险指标

| 指标 | 含义 |
|------|------|
| Sortino Ratio | 只惩罚下跌波动 |
| Maximum Drawdown | 从峰值到谷底最大跌幅 |
| Calmar Ratio | 年化收益/最大回撤 |
| VaR (95%) | 95%置信度下的最大损失 |

---

## 4.6 指标组合：从信号到特征

### 单一指标的局限

| 策略 | 胜率 | 年化收益（扣费后） |
|------|------|-----------------|
| MACD金叉/死叉 | 51% | -5%~+3% |
| RSI超买超卖 | 52% | -3%~+5% |
| 布林带突破 | 50% | -8%~+2% |

### 正确用法：特征工程

```
特征向量 = [
    RSI,
    RSI变化率,
    MACD_DIFF,
    MACD_柱状图变化率,
    布林带位置,
    ATR,
    成交量变化率,
    ...
]
→ 输入机器学习模型 → 输出：买入/卖出/持有 + 置信度
```

**多智能体视角**：
- Trend Agent：关注MACD、均线、ATR
- Mean Reversion Agent：关注RSI、布林带
- Regime Agent：关注波动率变化
- Risk Agent：关注VaR、最大回撤

## 代码实现

```python
import pandas as pd
import numpy as np

def calculate_indicators(df):
    # EMA
    df['EMA12'] = df['close'].ewm(span=12).mean()
    df['EMA26'] = df['close'].ewm(span=26).mean()

    # MACD
    df['MACD_DIFF'] = df['EMA12'] - df['EMA26']
    df['MACD_DEA'] = df['MACD_DIFF'].ewm(span=9).mean()
    df['MACD_Histogram'] = 2 * (df['MACD_DIFF'] - df['MACD_DEA'])

    # RSI
    delta = df['close'].diff()
    gain = delta.where(delta > 0, 0).rolling(14).mean()
    loss = (-delta.where(delta < 0, 0)).rolling(14).mean()
    rs = gain / loss.replace(0, np.nan)
    df['RSI'] = 100 - 100 / (1 + rs)
    df['RSI'] = df['RSI'].fillna(100)

    # Bollinger Bands
    df['BB_Middle'] = df['close'].rolling(20).mean()
    df['BB_Std'] = df['close'].rolling(20).std()
    df['BB_Upper'] = df['BB_Middle'] + 2 * df['BB_Std']
    df['BB_Lower'] = df['BB_Middle'] - 2 * df['BB_Std']
    df['BB_Position'] = (df['close'] - df['BB_Lower']) / (df['BB_Upper'] - df['BB_Lower'])

    # ATR
    high_low = df['high'] - df['low']
    high_close = (df['high'] - df['close'].shift()).abs()
    low_close = (df['low'] - df['close'].shift()).abs()
    tr = pd.concat([high_low, high_close, low_close], axis=1).max(axis=1)
    df['ATR'] = tr.rolling(14).mean()

    return df

def calculate_sharpe_ratio(returns, risk_free_rate=0.04, periods_per_year=252):
    excess_returns = returns - risk_free_rate / periods_per_year
    return np.sqrt(periods_per_year) * excess_returns.mean() / returns.std()
```

## 本课要点回顾

- 技术指标的本质：价格/成交量的数学变换，不创造新信息
- 掌握MACD、RSI、布林带、ATR的计算逻辑与适用场景
- 背离分析有价值，但需结合其他维度验证
- 用Sharpe、Sortino、Calmar等指标评估策略表现
- 单一指标局限明显，组合特征输入模型才能发挥真正价值
