## LTCM：诺贝尔奖得主的致命假设

1998年，LTCM拥有两位诺贝尔经济学奖得主和1250亿美元资产，核心假设是"市场收益服从正态分布"。俄罗斯债务违约后，单日亏损5.5亿，一个月亏损46亿——几乎全部资本。

失败原因：
- 市场是厚尾分布，极端事件频率远超预测
- 相关性会突变：危机时所有资产相关性飙升到1
- 波动率会聚集：大跌之后往往还是大跌

## 3.1 时间序列基础

### 序列 vs IID 假设

传统统计假设数据是**IID（独立同分布）**，但金融数据几乎从不满足：

```python
import numpy as np

# IID 数据：抛硬币
coin_flips = np.random.choice([0, 1], size=100)

# 金融数据：今天的价格依赖昨天
prices = [100]
for _ in range(99):
    prices.append(prices[-1] * (1 + np.random.normal(0, 0.02)))
```

### 滞后与自相关

**自相关**：当前值与过去值的相关性。

| 今天 (t) | 昨天 (t-1) |
|---------|-----------|
| -1% | +2% |
| +3% | -1% |
| +2% | +3% |
| -2% | +2% |

- 自相关接近0 → 随机游走
- 自相关正 → 动量效应
- 自相关负 → 均值回归

SPY日收益自相关约为0.02，接近随机游走。

```python
import pandas as pd

def calculate_autocorrelation(series, lag=1):
    return series.autocorr(lag=lag)

returns = prices_series.pct_change().dropna()
print(f"滞后1期自相关: {calculate_autocorrelation(returns, 1):.3f}")
print(f"滞后5期自相关: {calculate_autocorrelation(returns, 5):.3f}")
```

### 平稳性检验

```python
from statsmodels.tsa.stattools import adfuller

def check_stationarity(series, name="Series"):

    print(f"{name}:")
    print(f"  ADF 统计量: {result[0]:.4f}")
```

## 3.2 收益的数学定义

### 简单收益 vs 对数收益

AAPL从$100涨到$110：

```
简单收益 = (110 - 100) / 100 = 10%
对数收益 = ln(110/100) = ln(1.1) ≈ 9.53%
```

可加性对比：

| | Day 1 | Day 2 | 直接相加 |
|--|-------|-------|---------|
| 简单收益 | +10% | -9.09% | 0.91% ❌ |
| 对数收益 | +9.53% | -9.53% | 0% ✅ |

| 特性 | 简单收益 | 对数收益 |
|------|---------|---------|
| 可加性 | ✗ | ✓ |
| 对称性 | ✗ | ✓ |
| 正态假设 | 较难满足 | 更接近正态 |

### 累计收益计算

连续5天：+1%, -2%, +3%, +1%, -1%

**简单收益**（连乘）：
```
1.01 × 0.98 × 1.03 × 1.01 × 0.99 - 1 ≈ 1.94%
```

**对数收益**（相加）：
```
0.01 + (-0.02) + 0.03 + 0.01 + (-0.01) = 2%
```

### 年化收益率

60个交易日赚了5%，年化：
```
年化收益 = (1 + 5%)^(252/60) - 1 = 1.05^4.2 - 1 ≈ 22.7%
```

美股一年约252个交易日。

```python
def cumulative_return(returns, method='simple'):
    if method == 'simple':
        return (1 + returns).prod() - 1
    else:
        return returns.sum()

def annualize_return(total_return, days, trading_days_per_year=252):
    years = days / trading_days_per_year
    return (1 + total_return) ** (1 / years) - 1

ann_return = annualize_return(0.05, 60)
print(f"年化收益率: {ann_return:.2%}")
```

## 3.3 风险的数学定义

### 方差与标准差

5天日收益率：+1%, -2%, +3%, 0%, -1%

```
均值 = (1 - 2 + 3 + 0 - 1) / 5 = 0.2%
方差 = [0.64 + 4.84 + 7.84 + 0.04 + 1.44] / 5 = 2.96 (%²)
日波动率 = √2.96 ≈ 1.72%
年化波动率 = 1.72% × √252 ≈ 27.3%
```

| 年化波动率 | 典型资产 | 68%概率波动范围 |
|-----------|---------|--------------|
| 15% | SPY | -15%到+15% |
| 30% | TSLA | -30%到+30% |
| 80% | BTC | -80%到+80% |

**记忆公式**：年化波动率 = 日波动率 × √252 ≈ 日波动率 × 16

```python
def calculate_volatility(returns, annualize=True, trading_days=252):
    daily_vol = returns.std()
    if annualize:
        return daily_vol * np.sqrt(trading_days)
    return daily_vol
```

### 协方差与相关性

| 相关系数 | 含义 | 例子 |
|---------|------|------|
| +1 | 完全同步 | AAPL vs MSFT |
| 0 | 无关 | 黄金 vs 科技股 |
| -1 | 完全相反 | 股票 vs VIX |

危机时期相关性会飙升，这是LTCM失败的核心原因。

```python
def analyze_correlation(returns_a, returns_b):
    corr = returns_a.corr(returns_b)
    print(f"相关系数: {corr:.3f}")
    if corr > 0.7:
        print("→ 高度正相关：涨跌同步，分散效果差")
    elif corr < -0.3:
        print("→ 负相关：天然对冲，适合组合")
    else:
        print("→ 低相关：有分散风险的价值")
```

### 分布假设的危险性

SPY年化波动率20%，日波动率约1.26%。正态分布预测 vs 实际：

| 日期 | 事件 | 跌幅 | 几σ事件 | 理论多少年一遇 |
|------|------|------|---------|------------|
| 2020-03-16 | COVID崩盘 | -12.0% | 9.5σ | 10^20年 |
| 2008-10-15 | 金融危机 | -9.0% | 7.1σ | 10^12年 |
| 2011-08-08 | 美国降级 | -6.7% | 5.3σ | 1500万年 |
| 2018-02-05 | 波动率崩盘 | -4.1% | 3.3σ | 4年 |

## 3.4 金融时间序列的特殊性

### 厚尾分布 (Fat Tails)

| 指标 | 正态分布 | SPY实际 | 含义 |
|------|---------|---------|------|
| 偏度 | 0 | -0.7 | 暴跌更剧烈 |
| 峰度 | 3 | 8 | 极端事件频繁 |

```python
def analyze_tail_risk(returns):
    skew = returns.skew()
    kurt = returns.kurtosis()
    print(f"偏度: {skew:.3f} {'(负偏，小心暴跌)' if skew < -0.5 else ''}")
    print(f"峰度: {kurt+3:.3f} {'(厚尾！)' if kurt > 0 else ''}")
```

### 波动聚集 (Volatility Clustering)

2020年3月示例：
- 3月9日：SPY跌7.6%
- 3月12日：SPY跌9.5%
- 3月16日：SPY跌12.0%

波动率自相关高达0.7-0.9，而收益率自相关约为0。这意味着**波动率是可预测的**，收益率几乎不可预测。

```python
def detect_volatility_clustering(returns, window=20):
    rolling_vol = returns.rolling(window).std()
    vol_autocorr = rolling_vol.autocorr(lag=1)
    print(f"波动率的滞后1期自相关: {vol_autocorr:.3f}")
```

### 非平稳性与结构性断裂 (Regime Shift)

| Regime | 特征 | 适合策略 |
|--------|------|---------|
| 牛市 | 低波动、正收益、低相关 | 动量、买入持有 |
| 震荡 | 中波动、收益接近0 | 均值回归 |
| 危机 | 高波动、负收益、高相关 | 减仓、对冲 |

结构性断裂案例（近似值）：

| 时间 | 事件 | 波动率变化 | 相关性变化 |
|------|------|-----------|----------|
| 2008.09 | 雷曼倒闭 | 15%→80% | 0.3→0.95 |
| 2020.03 | COVID爆发 | 12%→85% | 0.4→0.90 |
| 2022.01 | 美联储加息 | 15%→30% | 0.5→0.70 |

```python
def detect_regime_change(returns, window=60):
    rolling_vol = returns.rolling(window).std()
    vol_change = rolling_vol.diff().abs()
    threshold = vol_change.quantile(0.95)
    regime_changes = vol_change > threshold
    print(f"检测到 {regime_changes.sum()} 个潜在的 Regime 变化点")
    return regime_changes
```

## 3.5 常见误区

- **误区一**：正态分布足够描述市场（不对，真实市场是厚尾分布）
- **误区二**：波动率越低越安全（低波动可能是暴风雨前的平静）
- **误区三**：相关性稳定可预测（危机时相关性可能从0.3飙升到0.9）
- **误区四**：历史数据能预测未来（Regime Shift后过去规律可能失效）
