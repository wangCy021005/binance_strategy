# 币安量化策略 — Memory Bank

## 项目背景
从A股量化转型，原因：
- A股动量IC=-0.02（短期反转，动量策略有害）
- A股量化依赖新闻/政策信号，纯价格策略难超越Buy&Hold
- A股T+1、无做空、涨停板等限制

## 币安优势（实测/研究依据）
- 动量IC=+0.05（中期趋势有效，3-10天）
- 24/7交易，无T+1
- 做空：永续合约，无额外成本
- 资金费率：加密特有稳定Alpha源
- 数据：免费API，OHLCV + 资金费率 + OI
